# lineage-os

This shell serves as the development shell on NixOS or Nix-capable platforms to work with the Android / LineageOS development stack.

## Devices

So far, the devices currently, or in progress to be, supported include

- `Flo`/`Flox`: ASUS Nexus 7 (2013)

More information to come after completion of the LineageOS build for that device.
