# Language Server Protocol

This helps power various capabilities in editing and inspecting files during development.

The following LSP servers may be beneficial in various parts of development.

_Note_: All reference names are based on unstable as of 2026-01-19

## Nix

- nixd
- nil

## Web Technologies

- `tailwindcss-language-server` (Tailwind)
- `svelte-language-server` (Svelte)
- `astro-language-server` (Astro)
- `vue-language-server` (Vue)
- `elmPackages.elm-language-server` (Elm)
- `typescript-language-server` (TypeScript)
- `glas` (Gleam)
- `package-version-server` (package.json)
- `phpactor` (PHP)

# Text Editing

- `mdx-language-server` (MDX)
- `marksman` (Markdown)
- `mpls` (Markdown preview)
